# Product overview

## Who it is for

GlassBox is for people who want everyday structure without a punitive productivity system. It is designed to make small, repeatable actions and reflection easier to return to.

## Product philosophy

The guiding idea is **productivity without guilt**. Missed tasks or habits should not turn a day into a failure state. The experience aims to make the next useful action visible while leaving room for mood, reflection, and imperfect routines.

## Five connected areas

| Area | Role in the experience |
| --- | --- |
| Self | A daily overview with mood, reflection, and personal context. |
| Tasks | Lightweight task planning and completion. |
| Habits | Focused routines that can be revisited daily. |
| Journal | Notes, gratitude, mood entries, and reflection. |
| Reward | Gentle progression and a companion layer that makes consistency feel more tangible. |

## Supporting product layers

GlassBox includes HealthKit-based context, onboarding, localization, and a rewards-and-companion layer. These are intended to support the core experience rather than turn it into a scoreboard.

## Current status

The application is in TestFlight and preparing for App Store release. Product validation and release readiness are ongoing; no public release date is announced.
